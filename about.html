Tentang Saya

Perkenalkan nama saya Hanif Nur Fadhil. Saya adalah seorang guru Informatika SMP dari Kabupaten Kepahiang, Provinsi Bengkulu. Saya mengajar di SMP Negeri 2 Bermani Ilir.

Ketertarikan saya terhadap dunia pemrograman telah tumbuh sejak masa SMA. Saat itu, saya mendapat kesempatan untuk mewakili sekolah dalam ajang Olimpiade Sains Nasional (OSN) bidang Komputer. Pengalaman tersebut menjadi titik awal yang membuka jalan saya menuju dunia teknologi yang penuh tantangan sekaligus menyenangkan.

Waktu itu, saya belajar menggunakan bahasa pemrograman Pascal, yang menjadi fondasi awal dalam memahami logika dan struktur kode. Kini, saya ingin melangkah lebih jauh dengan mendalami JavaScript untuk pengembangan web, serta Python untuk pengembangan program komputer.

Saya percaya, setiap bahasa pemrograman yang dipelajari bukan sekadar alat, melainkan jembatan untuk terus berkembang dan berkontribusi dalam dunia pendidikan digital.
